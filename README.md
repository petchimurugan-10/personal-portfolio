# Personal Portfolio — Petchimurugan S

A dark-themed, cybersecurity-inspired personal portfolio built with **Next.js 16**, featuring smooth animations, a responsive design, and a functional contact form.

---

## Tech Stack

| Technology | Purpose |
|---|---|
| Next.js 16 (App Router) | Framework |
| TypeScript | Language |
| Tailwind CSS v4 | Styling |
| Framer Motion | Animations |
| Lucide React | Icons |
| Nodemailer | Contact form emails |
| Google Fonts (Inter + JetBrains Mono) | Typography |

---

## Features

-  Dark cybersecurity theme with neon cyan accents
-  Scroll-triggered Framer Motion animations
-  Contact form that sends real emails to Gmail
-  Resume PDF download
-  Fully responsive (mobile + desktop)
-  Light / dark mode toggle
-  Dedicated pages: `/projects`, `/about`, `/contact`

---

## Sections

- **Hero** — Typing animation, CTA buttons, social links
- **About** — Bio, education, achievements, certifications
- **Skills** — 5 color-coded skill categories
- **Experience** — Internship timeline
- **Projects** — WAF, Network Scanner with GitHub links
- **Contact** — Contact form + social links
